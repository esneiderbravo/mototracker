# MotoTracker — Feature Registry

> **Purpose:** Living document that tracks all implemented features and the planned roadmap. Update every time a feature is added, changed, or removed.

---

## Tech Stack

| Layer | Technology |
|---|---|
| Framework | Flutter (Dart) |
| State management | Riverpod |
| Backend / DB | Supabase (PostgreSQL + Auth + Storage) |
| AI | Groq (llama-3.1-8b-instant) |
| Navigation | go_router |
| i18n | slang (ES / EN) |
| Theme | Custom dark theme with ThemeTokens |

## 🗺️ Roadmap — Planned Features

> Suggested priorities. Order may change based on feedback.

### 🔧 Maintenance Tracking
- [ ] Log maintenance records per motorcycle (oil, brakes, chain, filters, etc.)
- [ ] Maintenance history with dates and mileage
- [ ] Upcoming maintenance reminders (by km or by date)
- [ ] **AI**: suggest pending maintenance based on current km and motorcycle model
- [ ] **AI**: estimate maintenance cost by make / model / year

### 📄 Legal Documents
- [ ] SOAT registry per motorcycle (create, edit, delete, history)
- [ ] SOAT expiry notifications (30, 15 and 5 days before)
- [ ] Technical-Mechanical Inspection (RTM) registry
- [ ] RTM expiry notifications
- [ ] Document history per motorcycle
- [ ] **AI**: smart grouped alerts for all documents expiring soon

### 🛣️ Routes
- [ ] Log completed rides (origin, destination, km covered, date)
- [ ] Interactive map with route visualization
- [ ] Photo gallery per route
- [ ] **AI**: suggest popular routes based on location and motorcycle type
- [ ] Share route with the community

### 👥 Community
- [ ] Public rider profile (nickname, motorcycles, routes)
- [ ] Activity feed (shared routes, achievements, maintenance logs)
- [ ] Motorcycle groups / clubs
- [ ] Follower system
- [ ] Events and group ride announcements

### 📊 Statistics & Insights
- [ ] Dashboard summary: total km, completed maintenance, expenses
- [ ] Mileage evolution chart
- [ ] Total maintenance spend per motorcycle / per period
- [ ] **AI**: usage pattern analysis and personalized recommendations

### 🔔 Notifications
- [ ] Push notifications for maintenance reminders
- [ ] Push notifications for document expiry
- [ ] Community activity notifications

### 🔎 Search & Comparison
- [ ] Search other motorcycles in the community by make / model
- [ ] Compare specs between two motorcycles
- [ ] **AI**: maintenance cost comparison between models

### 🛒 Marketplace / Directory (long term)
- [ ] Community-recommended workshop directory
- [ ] Insurance provider directory with SOAT price comparison
- [ ] Second-hand parts marketplace between users

## 📝 Update Convention

Every time a feature is implemented or changed:
1. Move the item from **Roadmap** to **Implemented** with `[x]`.
2. Optionally add an implementation date comment: `<!-- implemented: YYYY-MM-DD -->`.
3. Update the file structure section if applicable.
4. Commit with message: `docs: update FEATURES.md`.
